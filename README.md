SCENARIO 1 – USER-BASED COLLABORATIVE FILTERING

Problem Statement
Recommend movies to users based on similar users' preferences.

Dataset Link:https://www.kaggle.com/datasets/rajmehra03/movielens100k

This project uses User-Based Collaborative Filtering to recommend movies based on similar users’ preferences. A User-Item matrix and cosine similarity are used to predict ratings and generate recommendations. The model is evaluated using RMSE and MAE, showing good accuracy.

SCENARIO 2 – ITEM-BASED COLLABORATIVE FILTERING

Problem Statement
Recommend similar items (movies/products) based on user ratings.

Dataset Link:https://www.kaggle.com/datasets/rajmehra03/movielens100k

This project uses Item-Based Collaborative Filtering to recommend movies based on item similarity. An Item-User matrix and cosine similarity are used to generate stable recommendations. The model is evaluated using RMSE and Precision@K, showing good performance and scalability
